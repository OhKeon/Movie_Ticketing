# Movie_Ticketing

## 🖥️: 프로젝트 소개
Spring Boot 를 사용한 영화 예매 사이트 입니다.
 
## 🧑‍🤝‍🧑: 멤버구성
- 오건
- 고민석
- 정주현

## ⚙️: 개발환경
- Kotlin 1.9.22
- Spring Boot 3.2.3
- Gradle Kotlin DSL
- Hibernate / Spring JPA
- MariaDB 11.0.6   
- Thymeleaf
- WAS
- RDS database
